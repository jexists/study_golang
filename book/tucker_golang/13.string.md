## 문자열
- 문자 집합
- 큰따옴표("): 특수문자 동작, 한줄로 입력(\n사용해서 여러줄 표현)
- 백쿼트(`): 특수문자 동작X, 여러줄 입력 가능
- 표준 문자코드: UTF-8 (다국어 문자 지원)

### UTF-8
- 영문자, 숫자, 일부 특수문자 1바이트로 표현
- 그 외 문자 2~3 바이트로 표현
- ANSI코드와 변환 가능
- UTF-16: 한문자에 2바이트 고정 사용

### rune 타입
- 문자 하나를 표현하는 타입
- 4바이트 정수타입인 int 32타입의 별칭 타입
- len() 함수: 문자열 바이트 크기
- []rune 타입 변환: 문자 글자 수
